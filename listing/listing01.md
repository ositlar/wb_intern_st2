Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```
Ответ
```
Outupt:
[77 78 79]
При создании среза с помощью [:] число перед : - начало включительно, после : - конец не включительно
```
